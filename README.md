## Installation

### Avec Linux

```bash
# Créer un environnement virtuel
python3 -m venv venv

# Activer l'environnement virtuel
source venv/bin/activate

# Installer les dépendances
pip install -r requirements.txt
```

```bash

python -m venv venv

# Activer l'environnement virtuel
.\venv\Scripts\Activate

# Installer les dépendances
pip install -r requirements.txt
```
